## リスクと対応策

| リスク | 対応策 |
|-------|--------|
| **Gemini API の遅延・障害** | ダミーレスポンスで代替、スコア計算ロジックのテストを先行 |
| **売上データが取得できない** | ダミーデータで代替 |
| **画像認識の精度不足** | バーコード認識での代替、手動 ID 入力フォーム |
| **AWS 移行に手間取る** | ローカル動作デモで発表を乗り切る |
